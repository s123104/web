什麼是 Google Apps Script？

GAS 是 Google 提供的雲端開發平台，讓你用 JavaScript 自動化 Google 服務，並建立強大的 Web 應用程式

免費雲端運算

無需維護伺服器，Google 提供穩定的雲端環境執行你的程式碼

原生整合

直接存取 Gmail、Drive、Sheets 等 Google 服務，無需複雜的 API 認證

Web App 部署

完整設置教學

從零開始建立 GAS 專案並部署到 Github Pages

1

建立 Google Apps Script 專案

前往 script.google.com 建立新專案

Code.gs 
function doPost(e) {


// 解析接收到的數據
const data = JSON.parse(e.postData.contents);

// 處理邏輯
const response = {
success: true,
message: `哈囉 ${data.name}！`,
interest: data.interest,
timestamp: new Date().toISOString(),
recommendation: getRecommendation(data.interest)
};

// 回傳 JSON 響應
return ContentService
.createTextOutput(JSON.stringify(response))
.setMimeType(ContentService.MimeType.JSON);
}

function getRecommendation(interest) {
const recommendations = {
‘programming’: ‘推薦學習 JavaScript 和 Python’,
‘design’: ‘推薦掌握 Figma 和 Adobe 工具’,
‘data’: ‘推薦學習 SQL 和 Tableau’,
‘marketing’: ‘推薦學習 Google Analytics’
};

return recommendations[interest] || ‘持續學習新技能！’;
}
2

部署為 Web 應用程式

點擊「部署」→「新增部署作業」
類型選擇「網頁應用程式」
存取權限設為「任何人」
複製產生的 Web App URL
3

建立 Github Repository

在 Github 建立新的 Repository 並啟用 Pages

JavaScript 串接代碼 
// 替換為你的 GAS Web App URL


const GAS_URL = ‘https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec’;

async function sendToGAS(data) {
try {
const response = await fetch(GAS_URL, {
method: ‘POST’,
headers: {
‘Content-Type’: ‘application/json’,
},
body: JSON.stringify(data)
});


return await response.json();


} catch (error) {
console.error(‘錯誤:’, error);
return { success: false, error: error.message };
}
}
4

啟用 Github Pages

Repository → Settings → Pages
Source 選擇「Deploy from a branch」
Branch 選擇「main」
等待部署完成，取得 Pages URL
