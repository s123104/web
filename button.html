Silent Mode開關立體陰影效果解析

這個精緻的「Silent Mode」開關使用了多種高級CSS和3D變換技術，創造出驚人的立體感和真實的陰影效果。讓我們一步步分析這些技術是如何實現的。

1. 基本陰影與深度層次

立體效果的核心是通過多層次的陰影來創造深度感。就像是在黑色絨布上放了一個精緻的小盒子，盒子的邊緣有反光，而內部有陰影。

/* 多層次陰影示例 */ .toggle-switch { position: relative; background-color: #222; border-radius: 14px; box-shadow: 0 6px 10px rgba(0, 0, 0, 0.6), /* 外部投影 */ 0 2px 2px rgba(0, 0, 0, 0.3); /* 次級投影 */ overflow: hidden; } /* 頂部高光模擬金屬邊緣 */ .toggle-switch::before { content: ''; position: absolute; top: 0; left: 0; right: 0; height: 1px; background: linear-gradient(to right, rgba(255, 255, 255, 0), rgba(255, 255, 255, 0.1), rgba(255, 255, 255, 0)); }

2. 凹凸表面效果

要創造出UI元素的凹凸感，我們使用漸變色和巧妙的陰影。就像一塊金屬被壓出了凹槽或凸起的部分。

/* 內凹效果 - 邊緣亮，中間暗 */ .toggle-concave::before { content: ''; position: absolute; background: linear-gradient(135deg, rgba(0, 0, 0, 0.8) 0%, rgba(0, 0, 0, 0) 50%, rgba(255, 255, 255, 0.05) 100%); border-radius: 14px; } /* 凸出效果 - 中間亮，邊緣暗 */ .toggle-convex::before { content: ''; position: absolute; background: linear-gradient(135deg, rgba(255, 255, 255, 0.1) 0%, rgba(0, 0, 0, 0) 50%, rgba(0, 0, 0, 0.3) 100%); border-radius: 14px; }

3. 金屬質感

金屬質感是通過精細的漸變和局部的高光實現的。就像一個精心打磨的金屬表面，在不同角度有不同的反光效果。

Silent mode
/* 金屬質感 */ .toggle-metal { background: linear-gradient(to bottom, #333 0%, #2a2a2a 100%); box-shadow: 0 4px 8px rgba(0, 0, 0, 0.5), 0 -1px 0 rgba(255, 255, 255, 0.05); } /* 頂部金屬質感高光 */ .toggle-metal::after { content: ''; position: absolute; top: 0; left: 0; right: 0; height: 1px; background: linear-gradient(to right, rgba(255, 255, 255, 0), rgba(255, 255, 255, 0.1), rgba(255, 255, 255, 0)); }

4. 3D 翻轉動畫

用CSS 3D變換實現的翻轉效果能讓UI元素呈現出真實的立體感。這就像翻轉一個實體卡片一樣自然。

Silent mode
/* 3D翻轉基本設置 */ .parent-container { perspective: 1000px; /* 提供3D透視效果 */ } .toggle-container { transform-style: preserve-3d; /* 保持3D效果 */ transition: transform 0.6s cubic-bezier(0.175, 0.885, 0.32, 1.275); } /* X軸翻轉 */ .toggle-container.flip-x { transform: rotateX(180deg); } /* Y軸翻轉 */ .toggle-container.flip-y { transform: rotateY(180deg); } /* 背面設置 */ .toggle-back { position: absolute; backface-visibility: hidden; /* 隱藏元素的背面 */ transform: rotateY(180deg); }

5. 陰影翻轉效果

當元素翻轉時，陰影也需要相應變化，這會增強整體的3D效果。想象一下，當你翻轉一個實物時，它的陰影會如何變化。

實現陰影翻轉的關鍵技術點：

  1. 使用絕對定位的外部陰影元素
  2. 翻轉時調整陰影的大小、模糊度和透明度
  3. 使用相同的動畫曲線確保陰影和主體同步變化
/* 陰影元素 */ .shadow { position: absolute; width: 220px; height: 20px; background: radial-gradient( ellipse at center, rgba(0, 0, 0, 0.4) 0%, rgba(0, 0, 0, 0) 70% ); bottom: -25px; border-radius: 50%; filter: blur(5px); transition: all 0.6s cubic-bezier(0.175, 0.885, 0.32, 1.275); } /* 翻轉時的陰影變化 */ .toggle-container.flip-x .shadow, .toggle-container.flip-y .shadow { opacity: 0.2; transform: scale(0.8); filter: blur(8px); }
