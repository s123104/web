常見問題

解答使用 Cursor AI 時的常見疑問，幫助你更順暢地使用這個強大工具。

Cursor AI 如何與 VS Code 比較？

Cursor AI 是 VS Code 的分支，保留了 VS Code 的大部分功能，同時整合了先進的 AI 功能，如程式碼生成、智能重寫和程式碼庫查詢等。對於喜歡 VS Code 的開發者來說，過渡到 Cursor AI 幾乎沒有學習曲線。

Cursor Rules 檔案如何工作？

Cursor Rules 檔案（.cursorrules）定義了 AI 在生成程式碼時應遵循的指示，如風格指南、常用方法或架構決策。它讓你能夠定制 AI 行為，確保 AI 生成的程式碼符合專案需求，提高一致性並減少手動修正。

如何讓 AI 理解我的專案結構？

使用 @Docs 添加你的專案文檔，或使用 /Generate Cursor Rules 生成專案特定規則。Cursor AI 也會自動分析你的專案檔案，理解結構和依賴關係，從而提供更準確的建議和生成更符合專案風格的程式碼。

我可以重新映射鍵盤快捷鍵嗎？

可以，所有鍵盤綁定均可在「鍵盤快捷鍵設定」頁面重新映射。打開命令面板（⌘⇧P），輸入「鍵盤快捷鍵」，選擇「首選項：開啟鍵盤快捷鍵」即可自定義所有快捷鍵。
