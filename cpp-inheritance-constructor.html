C++ 多重繼承「兩派修正」Demo

版本 C –「靜音魚缸」策略 🐟

比喻：
把 Person 想成會「咕嚕咕嚕」冒泡的大魚缸，
Faculty、Student 都搬了一缸進房間；
我們留魚缸、關打氣馬達 → 拔掉 cout，泡泡聲不再出現。

程式碼（專業註解）

// Version C: Keep inheritance, mute Person's constructor
#include <iostream>
using namespace std;

/* Person 仍是公共基底，但建構子不再輸出 ---------------- */
class Person{
public:
    Person(int /*x*/){ /* 被呼叫，但靜音 */ }
};

/* Faculty & Student 照舊 public 繼承 Person ---------- */
class Faculty : public Person{
public:
    Faculty(int x):Person(x){
        cout << "Faculty::Faculty(int ) called" << endl;
    }
};
class Student : public Person{
public:
    Student(int x):Person(x){
        cout << "Student::Student(int ) called" << endl;
    }
};

/* TA 同時繼承，初始化清單順序 = 繼承列表順序 ---------- */
class TA : public Faculty, public Student{
public:
    TA(int x):Faculty(x), Student(x){
        cout << "TA::TA(int ) called" << endl;
    }
};

int main(){ TA ta1(128); }

預期輸出

Faculty::Faculty(int ) called
Student::Student(int ) called
TA::TA(int ) called

版本 A –「分家養魚」策略 🐠

比喻：
覺得大魚缸太吵？乾脆各買自己的小魚缸！
把 Faculty、Student 與 Person 完全脫鉤，
就不會冒泡兩次。

程式碼（專業註解）

// Version A: Remove inheritance, each class stands alone
#include <iostream>
using namespace std;

/* Person 保留，但此題不再被繼承 ------------------------ */
class Person{
public:
    Person(int x){ cout << "Person::Person(int ) called" << endl; }
};

/* Faculty / Student 變獨立 ----------------------------- */
class Faculty{
public:
    Faculty(int){ cout << "Faculty::Faculty(int ) called" << endl; }
};
class Student{
public:
    Student(int){ cout << "Student::Student(int ) called" << endl; }
};

/* TA 多重繼承獨立類別 ---------------------------------- */
class TA : public Faculty, public Student{
public:
    TA(int x):Faculty(x), Student(x){
        cout << "TA::TA(int ) called" << endl;
    }
};

int main(){ TA ta1(128); }

預期輸出

Faculty::Faculty(int ) called
Student::Student(int ) called
TA::TA(int ) called
