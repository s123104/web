2. 資料庫、欄位與資料型態大解密

狼：資料庫就像一個超大倉庫，每個表格是倉庫區塊，欄位就像各個儲物格，能存各種不同型態的資料。常見的大分類包括：

  1. 數值資料 (Numeric Data)：integer、float、money... 適合計數、金額、價格等。
  2. 字串資料 (Character & Strings Data)：儲存字元、符號、文字。
  3. 日期/時間資料 (Date/Time Data)：記錄日期或時間，如 Date、Time、Timestamp。
  4. 布林值 (Boolean Data)：True/False、Yes/No、1/0。
  5. 二進位字串 (Binary Data)：儲存圖片或檔案等二進位內容。

不同資料庫或版本，型態與名稱都可能稍微有差異。以下列出 MS SQL 與 MySQL 常用型態及適用場景：

狼：基本上大多數專案常用的型態就是字串、整數、日期、小數等，除非有特別原因，不太建議在資料庫存大型檔案（圖片、Word等），通常只存路徑即可，以免效能受損。

3. MySQL 速成教學

狼：MySQL 就像一個巨大的冰箱，專門存放各式食材（資料）。用對資料型態，就能讓你的「冰箱」完美運行！

3.1 連線資料庫 🔓 (Open the Fridge)

mysql -u root -p

以 root 管理員登入，輸入密碼後就能打開冰箱囉～

3.2 創建資料庫 🏠 (Set Up the Kitchen)

CREATE DATABASE Kitchen;
USE Kitchen;

建立名為 Kitchen 的分區，放好之後就能擺放資料表～

3.3 建立資料表 🗄️ (Install Shelves)

CREATE TABLE Ingredients (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(50),
    quantity INT,
    expiry DATE
);

在 Kitchen 中建立 Ingredients 表，指定好資料型態，像是在冰箱中裝上層板。

3.4 新增資料 ➕ (Insert Items)

INSERT INTO Ingredients (name, quantity, expiry)
VALUES ('Eggs', 12, '2025-02-28');

丟入 12 顆雞蛋，並設定有效期限。

3.5 查詢資料 🔍 (Check Inventory)

SELECT * FROM Ingredients;

打開冰箱，檢查裡面的食材狀態。

3.6 更新資料 ✏️ (Adjust Stock)

UPDATE Ingredients SET quantity = 10 WHERE name = 'Eggs';

把雞蛋數量從 12 改成 10，補貨或修改都便利～

3.7 刪除資料 🗑️ (Discard Expired Items)

DELETE FROM Ingredients WHERE name = 'Eggs';

丟掉過期食材，要小心 WHERE，不然一不小心整個冰箱都砸光了～

4. MS SQL 速成教學

狼：MS SQL 就像一座高級金庫，功能強大、操作略有不同，一樣能讓你玩轉資料！

4.1 連線資料庫 🔓 (Open the Safe)

sqlcmd -S your_server -U your_username -P your_password

透過 sqlcmd 工具連線 MS SQL，打開金庫！

4.2 創建資料庫與資料表 🏠 (Set Up the Vault)

CREATE DATABASE Kitchen;
GO
USE Kitchen;
GO
CREATE TABLE Ingredients (
    id INT PRIMARY KEY IDENTITY(1,1),
    name NVARCHAR(50),
    quantity INT,
    expiry DATE
);
GO

利用 IDENTITY(1,1) 產生自動編號，NVARCHAR 支援 Unicode，確保多語言環境！

4.3 PHP 與 MS SQL 操作範例

以下以 PHP 連線並操作 MS SQL 為例，展示常見功能：

查詢 Query Data 🔍

<?php
$query = "SELECT * FROM Ingredients";
$stmt = sqlsrv_query($conn, $query);
while($row = sqlsrv_fetch_array($stmt, SQLSRV_FETCH_ASSOC)) {
    print_r($row);
}
?>

新增 Insert Data ➕

<?php
$query = "INSERT INTO Ingredients (name, quantity, expiry) VALUES ('Milk', 5, '2025-03-01')";
$stmt = sqlsrv_query($conn, $query);
?>

更新 Update Data ✏️

<?php
$query = "UPDATE Ingredients SET quantity = 8 WHERE name = 'Milk'";
$stmt = sqlsrv_query($conn, $query);
?>

刪除 Delete Data 🗑️

<?php
$query = "DELETE FROM Ingredients WHERE name = 'Milk'";
$stmt = sqlsrv_query($conn, $query);
?>
