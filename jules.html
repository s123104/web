日常開發任務

  • // 將 {特定} 檔案從 {x} 重構為 {y}...
    通用，適用於任何語言或程式庫。
  • // 新增測試套件...
    適用於缺乏測試覆蓋的程式庫。
  • // 為 {特定} Python 函數新增型別提示...
    適用於轉向型別化程式碼的 Python 專案。
  • // 為 {特定} 架構生成模擬資料...
    API、前端或測試密集型環境。
  • // 將這些 CommonJS 模組轉換為 ES 模組...
    JS/TS 專案現代化舊有程式碼。
  • // 將此基於回調的程式碼轉為 async/await...
    改進 JavaScript 或 Python 專案的非同步邏輯。
  • // 為此字典結構實現資料類...
    適用於使用 `dataclasses` 或 Pydantic 進行結構化資料處理的 Python 專案。

除錯

  • // 幫我修復 {特定} 錯誤...
    適用於任何遇到運行時或建置錯誤的程式庫。
  • // 為什麼 {這段特定程式碼} 執行緩慢？
    針對迴圈、函數或查詢的效能分析。
  • // 追蹤為什麼此值未定義...
    前端和後端 JS/TS 錯誤。
    • –
  • // 診斷此記憶體洩漏...
    伺服器端應用程式或長時間運行的進程。
  • // 新增日誌以幫助除錯此問題...
    適用於排查無聲失敗的情況。
  • // 尋找此非同步程式碼中的競爭條件
    JS、Python、Go 等並發系統。
  • // 在此 Python 腳本中新增列印語句以追蹤執行流程...
    用於除錯複雜 Python 腳本或理解意外行為。

文件撰寫

  • // 為此專案撰寫 README
    適用於缺乏基本專案概述的程式庫。
  • // 為此程式碼新增註解
    提高複雜邏輯的可維護性。
  • // 為此端點撰寫 API 文件
    REST 或 GraphQL 後端。
  • // 為此 Python 模組/類/函數生成 Sphinx 風格的文檔字符串...
    適用於使用 Sphinx 生成文件的 Python 專案。

測試

  • // 為此 API 端點新增整合測試
    Express、FastAPI、Django、Flask 應用程式。
  • // 撰寫模擬 fetch 的測試
    瀏覽器端的 fetch 或 axios 邏輯。
  • // 將此測試從 Mocha 轉換為 Jest
    JS 測試套件遷移。
  • // 為此函數生成基於屬性的測試
    功能性或邏輯繁重的程式碼。
  • // 在此測試套件中模擬慢速網絡條件
    網頁和行動應用程式。
  • // 撰寫測試以確保此函數的向後相容性
    函數庫或 SDK 維護者。
  • // 撰寫 Pytest 固定裝置以模擬此外部 API 調用...
    適用於使用 Pytest 並需要穩健模擬的 Python 專案。

套件管理

  • // 升級我的 linter 並自動修復破壞性配置變更
    使用 ESLint 或 Prettier 的 JS/TS 程式庫。
  • // 顯示 React 19 的變更日誌
    使用 React 的網頁前端應用程式。
  • // 哪些依賴項可以安全移除？
    臃腫或舊有程式碼庫。
  • // 檢查這些套件是否仍在維護
    注重安全或長期專案。
  • // 為自動更新設定 Renovate 或 Dependabot
    最適合具有 CI/CD 的活躍專案。

AI 原生任務

  • // 分析此程式庫並生成 3 個功能創意
    適用於初期或全新產品。
  • // 識別此檔案中的技術債
    具有混亂或脆弱邏輯的程式碼庫。
  • // 尋找跨檔案的重複邏輯
    缺乏 DRY 原則的龐大程式庫。
  • // 將相關函數分組並建議重構
    具有大量工具或輔助函數的專案。
  • // 幫我定義此問題的範圍，以便 Jules 解決
    與 Jules 合作解決實際問題。
  • // 將此函數轉換為可重用的插件/模組
    將邏輯繁重的程式碼模組化。
  • // 重構此 Python 函數以更適合並行處理...
    用於優化計算密集型 Python 應用程式的效能。

情境

  • // 根據最近的提交撰寫狀態更新
    管理與非同步溝通。
  • // 總結過去 7 天的所有變更
    休假後追趕進度。

趣味與實驗

  • // 在 {特定} 動作成功時新增彩紙動畫
    為前端網頁應用程式增添用戶喜悅時刻。
  • // 在 {特定} 建置完成時插入開發者笑話
    個人專案或團隊工具。
  • // 建立一個在終端機運行的小型 CLI 遊戲
    用於學習或社群趣味。
  • // 為此 UI 新增暗黑模式彩蛋
    設計繁重的網頁前端專案。
  • // 將此工具轉為 GitHub App
    可重用、平台整合的工具。

從頭開始

  • // 這個程式庫在做什麼？
    適合舊有程式庫或熟悉不熟悉的程式碼。
  • // 初始化一個啟用 CORS 的新 Express 應用程式
    使用 Node.js 和 Express 的網頁後端專案。
  • // 使用 Turborepo 和 PNPM 設定單一程式庫
    具有共享依賴的多套件 JS/TS 專案。
  • // 使用 Poetry 和 Pytest 啟動 Python 專案
    旨在實現乾淨依賴和測試設定的 Python 程式庫。
  • // 為 Chrome 擴充功能建立起始範本
    瀏覽器擴充功能開發。
  • // 我想建立一個網頁爬蟲—幫我開始
    使用 Python/Node 的資料爬取或自動化工具。
